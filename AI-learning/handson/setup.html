SETUP

ローカル環境セットアップガイド

演習ファイルを実行するための環境を準備しましょう

前提条件

演習を開始する前に、以下が揃っていることを確認してください。

  • Python 3.10 以上 — python --version で確認できます
  • pip（パッケージ管理） — pip --version で確認できます
  • Azure OpenAI Service または OpenAI API のアクセス権 — API キーと接続 URL が必要です
注意： Python のバージョンが 3.10 未満の場合、一部のライブラリが正常に動作しないことがあります。 python --version で確認し、必要に応じてアップグレードしてください。

セットアップ手順

Step 1

必要なライブラリのインストール

以下のコマンドで必要なライブラリをインストールしてください。

pip install openai pyyaml
Module 205（MCP）を受講する場合は追加で： 
pip install mcp
Step 2

config.yaml の作成

API 接続情報を設定するための config.yaml を作成します。

まず、サンプルファイルをダウンロードしてください：

config.yaml.example ⬇

次に、ダウンロードしたファイルを config.yaml という名前でコピーします：

cp config.yaml.example config.yaml

テキストエディタで config.yaml を開き、BASE_URL と API_KEY を実際の値に書き換えてください：

base_url: "https://your-resource.openai.azure.com/openai/deployments/your-model"
api_key: "your-api-key-here"
model: "gpt-4o"
セキュリティに関する注意： config.yaml には API キーが含まれます。 Git リポジトリにコミットしないよう注意してください。 .gitignore に config.yaml が含まれていることを確認してください。
Step 3

Python ファイルのダウンロード

各モジュールの演習ファイルは、それぞれのモジュールページからダウンロードできます。

トップページから受講するモジュールを選択し、「Handson」タブから演習ファイルをダウンロードしてください。

ダウンロードしたファイルは handson/ フォルダ（config.yaml と同じ場所）に配置してください。
Step 4

演習の実行

演習ファイルは必ず handson/ ディレクトリから実行してください。

cd /path/to/AI-learning/handson
python 201_openai-sdk.py --step 1
重要： handson/ ディレクトリ内に config.yaml と utils/ フォルダが存在する必要があります。 別のディレクトリから実行するとエラーが発生します。

実行ディレクトリについて

重要：必ず handson/ ディレクトリから実行してください

演習ファイルは handson/ ディレクトリに config.yaml と utils/ フォルダが存在することを前提に動作します。 別のディレクトリから実行すると、設定ファイルが見つからずエラーになります。

正しい実行例

cd /path/to/AI-learning/handson && python 201_openai-sdk.py --step 1

handson/ ディレクトリ構成

handson/
├── config.yaml              # API 接続設定（自分で作成）
├── config.yaml.example      # 設定テンプレート
├── utils/                   # 共通ユーティリティ（配布済み）
├── 201_openai-sdk.py        # Module 201 演習ファイル
├── 202_cli-chat.py          # Module 202 演習ファイル
├── 203_introduction_of_embeddings.py
├── 203_rag-chat.py
├── 204_function-calling-step1.py
├── 205_mcp-step1.py
└── ...                      # その他の演習ファイル

トラブルシューティング

ModuleNotFoundError: No module named 'utils'
handson/ ディレクトリから実行していることを確認してください。 config.yaml と utils/ フォルダが同じ場所に存在する必要があります。 
cd /path/to/AI-learning/handson
python 201_openai-sdk.py --step 1
API Key エラー（AuthenticationError）
config.yaml の api_key と base_url を確認してください。 Azure OpenAI の場合はリソース名・デプロイ名・APIバージョンが正確に設定されているかチェックしてください。 
base_url: "https://<resource-name>.openai.azure.com/openai/deployments/<deployment-name>"
api_key: "<your-api-key>"
BadRequestError: Request too large（Embeddings エラー）
Embeddings API に渡すテキストが長すぎる場合に発生します。 入力テキストを短くするか、複数のチャンクに分割して処理してください。 一般的に、text-embedding-3-small モデルは 8,191 トークンまで受け付けます。
上記で解決しない場合は、研修担当者または AI 推進部（ai-support@jxxx.co.jp / 内線 1234）にお問い合わせください。
