Step 1

テキストをベクトル化してコサイン類似度を計算する

Embeddings API を使って複数のテキストをベクトル化し、クエリに最も近い文書を検索してください。

# Step 1: Embeddings の取得とコサイン類似度計算
import numpy as np

documents = [
    "有給休暇の申請は人事システムから行えます。年間20日付与されます。",
    "テレワーク手当は月8,000円が支給されます。申請は翌月15日締切です。",
    "社内システムの障害は#incidentチャンネルに報告してください。",
    "経費精算の上限は1件あたり50,000円です。領収書の添付が必須です。",
]
query = "テレワークの手当について教えてください"

# 全文書をベクトル化
doc_response = client.embeddings.create(  # EDIT HERE!
    model=EMBEDDING_MODEL,
    input=documents
)
doc_embeddings = [item.embedding for item in doc_response.data]

# クエリをベクトル化
query_response = client.embeddings.create(  # EDIT HERE!
    model=EMBEDDING_MODEL,
    input=[query]
)
query_embedding = query_response.data[0].embedding

# コサイン類似度を計算して最も近い文書を探す
def cosine_similarity(a, b):
    a, b = np.array(a), np.array(b)
    return float(np.dot(a, b) / (np.linalg.norm(a) * np.linalg.norm(b)))

similarities = [cosine_similarity(query_embedding, doc_emb) for doc_emb in doc_embeddings]
top_idx = int(np.argmax(similarities))
print(f"最も類似した文書 (score={similarities[top_idx]:.4f}):")
print(documents[top_idx])
ヒント ①
client.embeddings.create(model=EMBEDDING_MODEL, input=texts)
input には文字列リストを渡します（複数同時にベクトル化可能）
結果の取り出し: response.data[i].embedding
ヒント ②
コサイン類似度 = 内積 / (ノルムa × ノルムb)
NumPy: np.dot(a, b) / (np.linalg.norm(a) * np.linalg.norm(b))
最大値のインデックス: np.argmax(similarities)

Step 2

ベクターストアを構築して RAG チャットを実装する

Axxx Japanの FAQ 文書をベクターストアに登録し、ユーザーの質問に対して関連文書を検索してからLLMに回答させてください。

# Step 2: ベクターストア構築と RAG チャット
import json

# FAQ 文書（実際の演習ではファイルから読み込みます）
faq_documents = [
    {"id": "hr-001", "source": "人事規程", "text": "有給休暇は年間20日付与されます。人事システムから申請し、上長承認が必要です。"},
    {"id": "hr-002", "source": "テレワーク規程", "text": "テレワーク手当は月8,000円支給。申請は翌月15日締切で、勤怠システムから申請。"},
    {"id": "it-001", "source": "IT運用規程", "text": "システム障害発生時はSlack #incidentに即時報告。重大障害はオンコール担当者に電話連絡。"},
    {"id": "fin-001", "source": "経費精算規程", "text": "経費精算は1件50,000円まで。領収書添付必須。月末締め、翌月10日払い。"},
    {"id": "fin-002", "source": "経費精算規程", "text": "交通費は実費精算。タクシーは深夜22時以降または1万円以上の案件で認可が必要。"},
]

# ベクターストアの構築
embed_response = client.embeddings.create(  # EDIT HERE!
    model=EMBEDDING_MODEL,
    input=[doc["text"] for doc in faq_documents]
)
store_embeddings = [item.embedding for item in embed_response.data]

def retrieve(query: str, top_k: int = 2) -> list[dict]:
    # クエリのベクトル化
    q_resp = client.embeddings.create(model=EMBEDDING_MODEL, input=[query])  # EDIT HERE!
    q_emb = np.array(q_resp.data[0].embedding)
    # 類似度計算とソート
    scores = [cosine_similarity(q_emb, np.array(e)) for e in store_embeddings]
    top_indices = np.argsort(scores)[::-1][:top_k]
    return [faq_documents[i] for i in top_indices]

def rag_answer(user_question: str) -> str:
    # 関連文書を検索
    retrieved = retrieve(user_question)
    context = "\n\n".join([f"[{d['source']}]\n{d['text']}" for d in retrieved])
    sources = "、".join([d["source"] for d in retrieved])

    # RAGプロンプト（コンテキスト注入）
    prompt = f"""  # EDIT HERE!
以下のAxxx Japanの社内文書を参照して質問に答えてください。
文書にない情報は「文書に記載がありません」と答えてください。
回答の末尾に「参照: {sources}」を必ず含めてください。

## 参照文書
{context}

## 質問
{user_question}
"""

    response = client.chat.completions.create(
        model=MODEL,
        messages=[{"role": "user", "content": prompt}]
    )
    return response.choices[0].message.content

# テスト
test_questions = [
    "テレワーク手当の申請方法と金額を教えてください",
    "タクシーを使う場合のルールは？",
]
for q in test_questions:
    print(f"\n❓ {q}")
    print(rag_answer(q))
ヒント ①
全文書を一度に client.embeddings.create(input=[...]) でベクトル化できます
input に各文書の "text" フィールドのリストを渡します
ヒント ②
プロンプトに「文書にない情報は答えない」「引用元を明記する」の2点を含めましょう
コンテキスト: [文書名]\n文書テキスト の形式で複数文書を結合します
